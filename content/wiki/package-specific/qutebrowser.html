Intro

This is my first entree to the package-specific wiki.

Qutebrowser is perhaps my favorite application that I use everyday. It is extremely efficient to use, highly configurable, and quite minimal (though it is written in Python).

What Qutebrowser is

Qutebroweser is a keyboard driven web browser, written and configured it Python.

See qutebrowser.org.

Installing and Running

Distribution's Main Package Manager

Qutebrowser is available in the repositories of most package managagers. For example, to install with the apk package manager, run: 

apk add qutebrowser # as root

Qutebrowser should be in the $PATH as qutebrowser, meaning that you can execute Qutebrowser simply with the command qutebrowser.

Flatpak

Flatpak will need to be installed, and the Flathub repository will need to be added.

For instructions on setting up flatpak, see flatpak.org/setup.

Installing

Meathod 1: Installing with the App ID

To install Qutebrowser as a flatpak by specifying the Application ID, run the following command: 

# System installation
flatpak install --user org.qutebrowser.qutebrowser # as root if not in `wheel` group

# User installation
flatpak install org.qutebrowser.qutebrowser
Meathod 2: Installing with a flatpakref File

Another meathod of installation is to install with a flatpakref. First, one must download the flatpakref file. You can do that by going to Qutebrowser page on the Flathub site and clicking the Install button, or by running the following command (in the directory that you want the flatpakref to be installed to). 

wget https://dl.flathub.org/repo/appstream/org.qutebrowser.qutebrowser.flatpakref

See the wget page.

Now you can run the flatpak command on the file. 

# System installation
flatpak install org.qutebrowser.qutebrowser.flatpakref # as root if not in `wheel` group

# User installation
flatpak install --user org.qutebrowser.qutebrowser.flatpakref

If you installed the flatpakref through your web browser and cannot find it run: 

find / -name org.qutebrowser.qutebrowser.flatpakref

That will return the file location of the flatpakref.

Meathod 3: Installing with a GUI

To install Qutebrowser as a flatpak through a GUI, the propor plugin package(s) need to be installed. For example to install the plugins with the apk package manager run the following commands: 

# To install for Gnome Software
apk add gnome-software-plugin-flatpak # as root

# To install for KDE Discover
apk add discover-backend-flatpak # as root

Once again, for details on setup for various distributions, see flatpak.org/setup

Now Qutebrowser should be available as a flatpak through the Gnome Software or KDE Discover GUI package managers.

Running

To run qutebrowser, you can do one of two things.

Meathod 1: Putting Qutebrowser in the $PATH

One meathod is to add the executable to the $PATH by first locating the executable. The executable should be in /var/lib/flatpak/exports/bin if installed as a system flatpak or ~/.local/share/flatpak/exports/bin if installed as a user flatpak. Just to make sure, use the find command on the two directories. 

# System installation
flatpak install org.qutebrowser.qutebrowser # as root if not in `wheel` group

# User installation
flatpak install --user org.qutebrowser.qutebrowser

The output will be the location of the executable, (i.e /var/lib/flatpak/exports/bin/org.qutebrowser.qutebrowser). Create a soft link between the exectuable, and the new location in /usr/local/bin (or another location in the $PATH). 

ln -s /var/lib/flatpak/exports/bin/org.qutebrowser.qutebrowser /usr/local/bin/qutebrowser # as root

Now you can execute Qutebrowser with the command qutebrowser.

Meathod 2: Executing Qutebrowser by Using the flatpak Command

The other meathod is to simply execute Qutebrowser with the flatpak command. 

flatpak run org.qutebrowser.qutebrowser
